パートナー規約

TOKYOFLOWER 販売パートナー規約

第1条（目的）

本規約は、株式会社東京フラワー（銀座東京フラワー、以下「当店」）が運営する販売パートナー制度（以下「本制度」）に関し、販売パートナー（以下「パートナー」）と当店の間の権利義務を定めるものです。

第2条（パートナー登録）

1. パートナー登録を希望する方は、所定の申請フォームより申請を行ってください。

2. 当店は申請内容を審査し、承認または不承認を決定します。審査基準は開示いたしません。

3. 以下に該当する場合、登録をお断りまたは取り消すことがあります。

・申請内容に虚偽がある場合
・反社会的勢力に該当する、またはそのおそれがある場合
・過去に本規約に違反したことがある場合
・その他、当店が不適切と判断した場合

第3条（成果報酬）

1. パートナーは、専用URL経由でお客様が商品を購入した場合、成果報酬を受け取ることができます。

2. パートナー自身による購入も成果報酬の対象となります。

3. 報酬の対象金額は商品代金（税込）とし、送料・手数料は含みません。

4. 報酬は月末締め、翌月末日に指定の銀行口座へお振込みいたします。

5. 振込手数料は当店が負担いたします。

6. 月間の報酬合計が5,000円未満の場合は、翌月以降に繰り越しとなります。

第4条（パートナーランク制度）

1. パートナーには以下の4つのランクがあり、直近3ヶ月間の実績に基づき毎月1日に判定されます。

ランク 報酬率 達成条件（直近3ヶ月） キープ条件（直近3ヶ月）
レギュラー 5% 登録時の初期ランク -
🥈 シルバー 8% 紹介件数 5件以上 または 紹介売上 30,000円以上 紹介件数 3件以上 または 紹介売上 20,000円以上
🥇 ゴールド 12% 紹介件数 15件以上 または 紹介売上 100,000円以上 紹介件数 10件以上 または 紹介売上 70,000円以上
💎 プラチナ 15% 紹介件数 30件以上 または 紹介売上 300,000円以上 紹介件数 20件以上 または 紹介売上 200,000円以上

2. 達成条件を満たした場合、翌月1日よりランクアップします。

3. キープ条件を満たさなかった場合、1つ下のランクに降格します。ただし、レギュラー以下には下がりません。

4. 紹介件数・紹介売上はいずれかを満たせばランク条件を達成したものとします（「または」条件）。

5. 成果確定済み（出荷完了）の注文のみが集計対象です。キャンセル・返品分は含みません。

第5条（成果の確定）

1. 注文が出荷完了となった時点で成果が確定します。

2. 以下の場合、成果は無効となります。

・お客様が注文をキャンセルした場合
・商品が返品された場合
・不正な注文と当店が判断した場合

第6条（禁止事項）

パートナーは、以下の行為を行ってはなりません。

1. 虚偽または誇大な表現による商品の宣伝
2. スパム行為（大量の迷惑メール送信、掲示板への無差別投稿等）
3. 当店のブランドイメージを損なう行為
4. 第三者になりすます行為
5. 追跡情報の不正操作（Cookie操作、リダイレクト偽装等）
6. 架空の注文や返品を前提とした注文による報酬の不正取得
7. リスティング広告において「TOKYOFLOWER」「銀座東京フラワー」「ginzaTOKYOFLOWER」等の当社名称を使用し、当社が販売しているように見せかける行為
8. その他、当店が不適切と判断する行為

第7条（パートナー資格の停止・取消）

1. 前条の禁止事項に違反した場合、当店はパートナー資格を即時停止または取り消すことができます。

2. 本規約違反による資格取消の場合、違反行為が発生した月以降の未払い報酬は支払われないことがあります。

3. 不正に取得した報酬について、当店は返還を求めることができます。

第8条（個人情報の取扱い）

1. 当店は、パートナーの個人情報を本制度の運営目的でのみ使用します。

2. 法令に基づく場合を除き、第三者に提供することはありません。

3. 報酬の支払いに必要な範囲で、金融機関に口座情報を提供することがあります。

第9条（規約の変更）

1. 当店は、必要に応じて本規約を変更できるものとします。

2. 報酬率・支払条件など重要な変更を行う場合は、効力発生の14日前までにパートナーへ通知します。

3. 変更後の規約は、当店ウェブサイトに掲載した時点で効力を生じます。

4. 変更後もパートナー活動を継続した場合、変更に同意したものとみなします。

第10条（免責事項）

1. システム障害等により追跡ができなかった場合の成果について、当店の故意または重大な過失がある場合を除き、当店は責任を負いません。

2. パートナーの宣伝活動に起因するトラブルについて、当店は責任を負いません。

第11条（有効期間）

1. パートナー登録が承認された日から1年間とし、双方から解約の申し出がない場合、自動的に1年間延長されるものとします。

2. パートナーは、30日前までに当店に通知することで、いつでも解約できます。

第12条（合意管轄）

本規約に関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

附則：本規約は2026年3月12日より施行します。
