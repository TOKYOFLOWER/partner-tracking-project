# プロジェクト概要

## 目的
自社カート `cart.tokyoflower.jp` に販売パートナー紹介トラッキングを実装し、
`?id=ptn0001` のようなパラメータから紹介元を記録、注文・出荷完了・成果集計までを管理する。

## 初期方針
- まずは動く最小構成を作る
- 後から報酬計算、パートナー画面、QRコード機能を追加できるようにする
- GitHub 管理の UI でダッシュボードを確認できるようにする

## 初期リリースの要点
- 有効な partner_id を Cookie に 30 日保持
- 注文時に partner_id を保存
- 送状番号入力で出荷完了扱い
- 出荷完了で成果確定
- キャンセルは成果なし
